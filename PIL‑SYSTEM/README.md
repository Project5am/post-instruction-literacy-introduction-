# Post Instruction Literacy System

This repository contains the governed architecture for PIL as a practice.  
It defines the taxonomy, tiers, diagnostics, and governance kernels that maintain structural integrity across all instruments.

      Governance
      Versioning
      Integration

## Purpose

The system defines how PIL operates at the level of structure.  
It does not contain instruments.  
It governs them.

## Invariants

- Tiers remain separate.  
- No role performs diagnostic or governance functions.  
- No stance condition generates or interprets material.  
- Diagnostics detect deviation but do not correct it.  
- Governance kernels enforce system level constraints.



## Additions

Any new structure must be evaluated against the taxonomy and the governance kernel before inclusion.
