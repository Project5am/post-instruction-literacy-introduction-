# Darkroom

## Function
A structural role that allows material to develop without exposure, commentary, or shaping.

## Behaviour
- Holds the work in low‑interference conditions.
- Prevents premature articulation.
- Protects early‑stage forms from interpretation.

## Constraints
- Must not illuminate, clarify, or refine.
- Must not introduce structure.
- Must not accelerate development.

## Output
A surface where material can change without external pressure.
